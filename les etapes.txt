npm init -y : initialiser un projet Node.js en créant automatiquement un fichier package.json avec des valeurs par défaut.
 Explication rapide des nouveaux packages :
dotenv : Charge les variables d’environnement depuis un fichier .env.
express : Framework web pour Node.js.
mongoose : ODM (Object-Document Mapping) pour MongoDB.
passport : Middleware d’authentification pour Node.js.
passport-google-oauth20 : Authentification avec Google OAuth 2.0.
passport-local : Authentification avec nom d’utilisateur/mot de passe.
uuid : Génère des identifiants uniques universels (UUID).
zod : Librairie pour la validation des schémas de données.
bcrypt@5.1.1 : Librairie pour hacher les mots de passe de manière sécurisée.
cookie-session@2.1.0 : Middleware pour gérer les sessions avec des cookies en Node.js.
cors@2.8.5 : Middleware permettant de gérer les requêtes cross-origin (CORS).
